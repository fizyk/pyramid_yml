Introduction
============

**tzf.pyramid_yml** is a convenience package, that allows for a yaml settings,
that can be kept in a structured, clear way, and also extensible depending on an 'environment'.

Testing
-------

To run tests you'll need py.test installed. To install all dependencues
required for running tests, simply run `pip install tzf.pyramid_yml[tests]`.
After that, simply run `py.test`
